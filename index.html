Responsive Web Design - View Ports

<meta name="viewport" content="width=device-width, initial-scale=1.0">

The above meta element is added to the head of all html documents, the device-width sets the width of the Page to follow the screen-width of the device used to view the page. The Initial-scale sets the initial zoom when the page is forst loaded.

Roles Atrribute

The role attribute describes the role of an element in programs that can make use of it, such as screen readers or magnifiers.

<a href="#" role="button">Button Link</a>

Screen Readers will read this element as “button” instead of “link”.

aria labelling

'aria-label' is used to define a string that labels the current element, its best used in cases where the text labe is not visible on the screen. It is useful for assistive technologies to understand what elements of the screen are for. A button to close a dialogue box for example might not have the text close written on. using an aria-label screen readers will be able to determine that the button is used to 'close'.

Box-Sizing

The box-sizing property determines how the width and the hieght of an element are calculated .By defualt any border or padding is added to the exhisting widht/ height of the element. If box-sizing is set to border-box on an element those properties are included in the total value
