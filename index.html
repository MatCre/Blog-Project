Div wrappers

The div element defines a section in a HTML document. It is commonly used as a container for HTML elements

    Uses & Tips
  • a wrapper for the body of the html
  • to create sections of content that can be controlled layout wise using CSS gridbox.
  • By default, browsers always place a line break before and after the div element.
